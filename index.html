プログラム復習コンテンツ

画面をリフレッシュ

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(480, 120);

　　 　fill(0, 102);

　　 　noStroke();

　　 }

　　 void draw(){

　　 　background(204)

　　 　ellipse(mouseX, mouseY, 9, 9);

　　 }

説明文

　draw()関数の最初にbackground関数を入れることで、draw()が繰り返されるたびに画面がbackgroundで指定した色に塗りつぶされる。

　そのため、draw()の最初にbackgroundを入れることで画面のリフレッシュとして用いられる。
